# 通用方法

## 日期格式化属性

在甘特表的属性中，可以通过给定一个格式化字符串来获取指定格式的日期内容，它的全部参数如下：

| fotmat | description |
| :----: | :---------: |
|   y    |     年      |
|   q    |    季度     |
|   M    |     月      |
|   d    |     日      |
|   H    |     时      |
|   m    |     分      |
|   s    |     秒      |
|   S    |    毫秒     |
