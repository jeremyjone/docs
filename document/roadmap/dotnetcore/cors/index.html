# 配置 CORS

最简单的方式：

services.AddCors(options =>
{
    // 添加策略，名称需要和中间件使用的名称一致
    options.AddPolicy("allowedCors", policy =>
    {
        policy.WithOrigins("http://localhost:5000")
            .AllowAnyHeader()
            .AllowAnyMethod();
    });
});
