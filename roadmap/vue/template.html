# 项目模板

# 如何添加模板

# 方法一

通过 degit 安装：

首先安装 degit

npm install -g degit
1

然后可以下载模板。例如：

npx degit jeremyjone/<repertory_name> <project-name>
1

提示

方法一创建的项目模板不包含 git，需要进入目录后手动执行 git init 方可。

# 方法二

可以直接 clone 项目到本地即可。例如：

git clone https://github.com/jeremyjone/<repertory_name>.git
mv <repertory_name> <project-name>
1
2
